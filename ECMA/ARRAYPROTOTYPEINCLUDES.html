ARRAY.PROTOTYPE.INCLUDES

Includes is similar to indexOf, however it returns true or false instead the index.
Exemple:
let fruits = ['apple', 'pineapple', 'banana', 'guava', 'grape'];
fruits.includes('grape'); // true
fruits.includes('mango'); // false

Includes work with NaN (Not a number). It has also a second parameter that allows to perform the search from a certain index:
fruits.includes('grape', 0); // true
fruits.includes('grape', 1); // false
