OBJECT.VALUES / OBJECT.ENTRIES

This function is used to extract values or value pairs similar to Objects.key.

Object.values returns an array of enumerable property values of objects. And we can iterate over it using the good old Array.prototype.forEach combined with arrow functions of the implicit return ES6: // Object literal
let obj = { a: 1, b: 2, c: 3 };
Object.values(obj).forEach(val => console.log(val)); // 1, 2, 3

Object.entries, on the other hand, will return an array of object enumerable property key-values (such as an array). Each item in the resulting array will also be an array.
// Object literal
let obj = { a: 1, b: 2, c: 3 };
JSON.stringify(Object.entries(obj)); // '[["a",1],["b",2],["c",3]]'

// String
Object.entries("JS"); // [ [ '0', 'J' ], [ '1', 'S' ] ]
